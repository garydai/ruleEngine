## fabric

### 安装

```
pip install fabric

```

### 例子
fabfile.py文件中每个函数就是一个任务，任务名即函数名
```
from fabric.api import *

env.hosts = ['xxx.xxx.xxx.xxx']
env.user = 'dev'
env.port = 1022
env.key_filename = ['xxx.pem']

ip = '1.2.3.4'

def canary():
	sudo("echo -e 'ipcanarylist = {\"%s\"}\nipblacklist = {}' > /usr/local/openresty/nginx/conf/lua/config.lua" % (ip))
	

```
